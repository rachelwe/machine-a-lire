# Guide Utilisateur

![accueil](/doc/captures/accueil-admin.png)

**Guide Complet pour Ajouter et Gérer du Contenu**

Vous êtes gestionnaire de contenu ou locataire d'une **Machine à Lire / Makina Bellet**, une **borne interactive** de consultation et d'impression de textes (complets ou extraits). 

Ce guide vous explique comment ajouter et supprimer des articles sur la machine, pas à pas.

**Structure** :
```
AJOUT/
└── Collections/
    └── Autrices/
        ├── articles/
        │   ├── 001.md
        │   ├── 002.md
        └── images/
```

### [Partie 4: Clés USB et Configuration](./guide-utilisateurs-04-cles-usb.md)

- Clé USB AJOUT (ajouter du contenu)

- Configuration globale et par-collection

**Points clés** :
- **⚠️ OBLIGATION : Retirer la clé après traitement**
- AJOUT n'efface JAMAIS les fichiers existants
- Attention : l'utilisation de la clé SUPPR supprime tous les textes, prévoyez une sauvegarde de vos fichiers d'origine hors de la machine !
