# Documentation Machine à Lire V2

Bienvenue dans la section documentation! Vous trouverez ici tous les guides et ressources pour utiliser, gérer et administrer votre **Machine à Lire**.

---

## Résumé

### [Guide Utilisateurs](./guide-utilisateur/README.md)

Pour les **gestionnaires de contenu** et **locataires** d'une Machine à Lire.

Ce guide complet explique :
- Les fondamentaux du format Markdown
- Comment ajouter et gérer des articles
- La structure des métadonnées
- L'organisation des fichiers sur clé USB
- Les workflows complets
- Les réponses aux questions fréquentes

**6 parties** : Commencez par la Partie 1 pour comprendre les bases!

---

### [Documentation Technique](../README.md)

Pour les **administrateurs** et **développeurs** qui déploient ou modifient la machine.

Contient :
- Aperçu du projet et architecture
- Instructions d'installation et configuration
- Conseils d'administration

---

### [Dupliquer une Machine à Lire](./dupliquer-makina.md)

Guide technique complet pour **cloner ou dupliquer** une Machine à Lire existante.

Utile si vous devez :
- Créer une nouvelle machine basée sur une configuration existante
- Migrer des données vers une autre Raspberry Pi
- Mettre en place une machine identique sur un autre site

---

### [Fichiers d'Exemple](./exemples/)

Des templates et exemples prêts à l'emploi :

- **Config Collections** : Exemple de fichier de configuration pour une collection
- **Config Global** : Choix de l'imprimante, du texte de changement de collection, de la collection chargée par défaut
- **Tutoriel** : Exemple complet avec instructions d'utilisation

Utilisez ces fichiers comme point de départ pour vos propres configurations et textes !
